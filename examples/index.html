Properties vs Getters/Setters

对比

C# 属性与 Java getters/setters 的对比

Java 风格

1public class JavaDevice {
2    private String name;
3    private boolean active;
4    
5    public String getName() { return name; }
6    public void setName(String name) { this.name = name; }
7    
8    public boolean isActive() { return active; }
9    public void setActive(boolean active) { this.active = active; }
10}

C# 风格

1public class CSharpDevice 
2{
3    public string Name { get; set; } = string.Empty;
4    public bool IsActive { get; set; }
5    
6    // 计算属性
7    public string Status => IsActive ? "在线" : "离线";
8    
9    // 带验证的属性
10    private int _id;
11    public int Id 
12    { 
13        get => _id;
14        set => _id = value > 0 ? value : throw new ArgumentException("ID必须大于0");
15    }
16}

主要差异:

  • C# 属性语法更简洁
  • 支持自动实现的属性 { get; set; }
  • 支持计算属性（只读）
  • 支持初始化器

LINQ vs Java Streams

对比

数据查询操作的对比

Java Streams

1List<Device> activeDevices = devices.stream()
2    .filter(d -> d.isActive())
3    .sorted(Comparator.comparing(Device::getName))
4    .collect(Collectors.toList());

C# LINQ - 方法语法

1var activeDevices = devices
2    .Where(d => d.IsActive)
3    .OrderBy(d => d.Name)
4    .ToList();

C# LINQ - 查询语法

1var activeDevices = from d in devices
2                   where d.IsActive
3                   orderby d.Name
4                   select d;

优势对比:

  • LINQ 提供两种语法：方法语法和查询语法
  • 查询语法更接近 SQL，易于理解
  • 方法语法更函数式，可以链式调用

Java CompletableFuture vs C# async/await

Java CompletableFuture

1public CompletableFuture<Device> getDeviceAsync(int id) {
2    return CompletableFuture.supplyAsync(() -> {
3        return deviceRepository.findById(id);
4    });
5}
6
7// 调用异步方法
8deviceService.getDeviceAsync(1)
9    .thenAccept(device -> System.out.println(device.getName()));

C# async/await

1public async Task<Device> GetDeviceAsync(int id)
2{
3    var device = await _deviceRepository.FindAsync(id);
4    
5    if (device == null)
6        throw new DeviceNotFoundException("Device not found");
7        
8    return device;
9}
10
11// 调用异步方法
12try 
13{
14    var device = await deviceService.GetDeviceAsync(1);
15    Console.WriteLine(device.Name);
16}
17catch (DeviceNotFoundException ex)
18{
19    Console.WriteLine($"Error: {ex.Message}");
20}

C# 优势:

  • 语法更简洁，接近同步代码
  • 异常处理更直观
  • 编译器自动生成状态机
