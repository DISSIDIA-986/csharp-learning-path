macOS 开发环境搭建指南

macOS 开发环境搭建

0/6 完成

步骤 1: 验证 Homebrew 安装

确保您的Mac上已安装包管理器 Homebrew

终端命令
$brew --version

步骤 2: 安装 .NET SDK

通过 Homebrew 安装最新的 .NET SDK

终端命令
$brew install dotnet
$dotnet --version

步骤 3: 安装 VS Code 扩展

在 VS Code 中安装 C# 开发必需的扩展

必需扩展列表：

  • • C# Dev Kit (Microsoft) - 核心开发工具包
  • • REST Client (Humao) - API测试工具
  • • NuGet Gallery (pcislo) - 包管理器界面

步骤 4: 安装 Docker Desktop

为数据库开发准备容器化环境

终端命令
$brew install --cask docker

步骤 5: 安装数据库管理工具

安装 Azure Data Studio 用于数据库管理

终端命令
$brew install --cask azure-data-studio

步骤 6: 创建第一个项目

验证环境搭建是否成功

终端命令
$mkdir MyFirstDotNetApp
$cd MyFirstDotNetApp
$dotnet new webapi -n MyFirstApi
$cd MyFirstApi
$code .
$dotnet run
